# 🏠 高端租屋代管平台｜各角色註冊與加入流程提示詞

## 🎯 任務目標
設計一套清晰的用戶註冊與加入流程，支援四種角色（platform_admin / landlord / manager / tenant），並結合高端代管、多組織及權限控管。

---

## 👤 角色流程

### 1️⃣ 平台管理者（platform_admin）
- **建立方式**：由系統預設
- **流程**：
  1. 系統初始化時預設帳號
  2. 使用預設憑證或環境變數設定登入
  3. 登入後擁有全平台控制權限
- **備註**：不屬於任何 organization

---

### 2️⃣ 房東（landlord）
- **建立方式**：邀請制（平台管理者建立或邀請）
- **流程**：
  1. platform_admin 建立帳號或發送邀請
  2. 房東收到邀請 → 註冊 / 完成帳號啟用
  3. 系統自動建立一個 organization
  4. 登入後可管理自有資產與房源
- **附加**：
  - 房東可邀請 manager
  - 房東可審核租客加入租約

---

### 3️⃣ 代管（manager）
- **建立方式**：邀請制（房東或平台管理者邀請）
- **流程**：
  1. 房東或 platform_admin 發送邀請（含組織與角色）
  2. manager 點擊邀請 → 註冊 / 登入
  3. 自動加入指定 organization
  4. 登入後可管理授權房源
- **附加**：
  - 可加入多個 organization（不同房東指派）
  - 權限受組織範圍限制

---

### 4️⃣ 房客（tenant）
- **建立方式**：邀請制（僅能由房東或 manager 邀請給予邀請碼才能註冊，不可自行註冊）
- **流程**：
  1. 房東或 manager 生成邀請碼或發送租約邀請
  2. 房客憑邀請碼完成註冊並自動綁定帳號與房源
  3. 房客登入後僅可查看自己的租約與資訊
- **備註**：
  - 嚴格限制不可在無邀請之情況下自行註冊
  - 註冊即完成房源與租約綁定

---
